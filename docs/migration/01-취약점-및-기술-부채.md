# 라이브러리 취약점 및 기술 부채

> 작성일: 2026-05-25  
> 근거: `pom.xml` 직접 분석 + 알려진 CVE 패턴 (정기적으로 `./gradlew dependencyCheckAnalyze` 로 갱신 예정)

## 심각도 요약

| 등급 | 항목 수 (대표) | 대응 방향 |
|------|----------------|-----------|
| **Critical** | Log4j 1.x, Fastjson 1.2.x, 구 Spring Security | Boot 4 + 의존성 BOM 일괄 상향 |
| **High** | Jackson 2.8.x, Commons Collections 3.x, 구 Tomcat embed | 제거 또는 최신 패치 |
| **Medium** | MyBatis 3.1, Druid 1.0, POI 3.16, Solr 5.5 | 단계적 교체·격리 |
| **Structural** | XML Spring, WAR, AngularJS EOL | 아키텍처 마이그레이션 |

## 알려진 고위험 의존성

### Log4j 1.2.17

- **위험**: Log4Shell(CVE-2021-44228) 계열, 1.x EOL
- **현재**: `log4j:log4j:1.2.17` + `slf4j-log4j12`
- **목표**: Logback + SLF4J (Spring Boot 기본)

### Alibaba Fastjson 1.2.29

- **위험**: 다수 역직렬화 CVE (1.2.x 계열)
- **현재**: API/직렬화에 사용 가능성
- **목표**: Jackson 2.17+ (Boot BOM) 로 통일 후 Fastjson 제거

### Spring Framework 4.3 / Spring Security 4.1

- **위험**: OSS 지원 종료, 알려진 보안 패치 미적용
- **목표**: Spring Boot **4.0.6** + Spring Security **7.x**

### Jackson 2.8.1

- **위험**: CVE-2017-7525, CVE-2018-11339 등 다수
- **충돌**: `kylin-jdbc` → `avatica` 번들 Jackson (AGENTS.md Avatica 수동 패치)
- **목표**: Boot 관리 Jackson + Kylin/Avatica 의존성 재검토 또는 제거

### Commons Collections 3.2.1

- **위험**: 역직렬화 gadget 체인 (구버전)
- **목표**: 4.x 또는 사용처 제거

### H2 1.4.196

- **위험**: 구버전 RCE/권한 이슈 (버전별 CVE)
- **목표**: H2 2.x (Boot 4 호환) — 로컬·테스트 전용

### 기타 구버전

## 구조적 기술 부채

1. **XML 기반 Spring** — 타입 안전성·테스트 어려움  
2. **WAR + 외부 Tomcat** — Boot 내장 서버·컨테이너 표준화 필요  
3. **AngularJS 1.x (EOL)** — React + Vite 로 교체  
4. **다중 JSON 라이브러리** — Jackson / Fastjson / org.json  
5. ~~**테넌트 `business_code`**~~ — 제거됨 (단일 테넌트, Flyway V10·로그인 UI 정리)  
6. ~~**`SDII*` 명명**~~ → **`Legacy*DataJson`** 로 리네이밍 완료 (레거시 WAR)  
7. ~~**Maven HTTP 저장소** (`maven.aliyun.com`)~~ — **제거됨** (Gradle은 Maven Central만 사용)  
8. **테스트 부재** — 회귀 위험 높음 → TDD·계약 테스트 도입  
9. **`webapp/report/`** — API 없는 정적 신한 리포트 (삭제 완료)  
10. **Jasypt + 평문 dev 비밀** — Boot 4 `spring-boot-starter` + 환경변수/시크릿 관리로 이전
