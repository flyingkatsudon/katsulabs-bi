# 현재 기술 스택 (As-Is)

> 작성일: 2026-05-25  
> 대상: `katsulabs-bi` (레거시 BI 포크, 레거시 `bdp` WAR)

## 개요

본 문서는 **제거된 레거시 WAR** 기준 As-Is입니다. 당시 코드베이스는 오픈소스 BI 대시보드 포크의 **WAR 패키징** 웹 애플리케이션이었습니다. 빌드·배포는 **Maven + Tomcat 8.5** 중심이며, 프론트엔드는 **AngularJS 1.x + AdminLTE + JSP** 혼합 구조입니다.

## 백엔드

| 구분 | 버전 / 기술 | 비고 |
|------|-------------|------|
| 언어 | Java 8 (`jdk.version=1.8`) | `pom.xml` |
| 빌드 | Maven 3.x, `packaging=war` | `tomcat7-maven-plugin` 로컬 실행 |
| 프레임워크 | Spring Framework **4.3.7** | XML 기반 설정 (`spring-*.xml`) |
| 보안 | Spring Security **4.1.0** | JDBC 인증, `레거시 인증*` 필터 |
| 영속성 | MyBatis **3.1.1** + MyBatis-Spring **1.1.1** | XML Mapper |
| 메타 DB | MySQL 8 / H2 1.4.196 (프로필) | Docker Compose 또는 `-Denv=local` |
| 연결 풀 | Alibaba Druid 1.0.27 | |
| 캐시 | EhCache 3.2.0, Redis (Spring Data Redis 1.7.4) | |
| 스케줄 | Quartz 2.2.3 | |
| 로깅 | Log4j 1.2.17 + SLF4J 1.7.21 | |
| JSON | Jackson 2.8.1, Fastjson 1.2.29, org.json | 중복·구버전 혼재 |
| 기타 통합 | Kylin JDBC, Presto JDBC, Solr 5.5, Elasticsearch provider, POI 3.16 | |

## 프론트엔드

| 구분 | 기술 |
|------|------|
| SPA 프레임워크 | AngularJS 1.x (`src/main/webapp/` 레거시 SPA) |
| UI | AdminLTE, Bootstrap 3, jQuery 2.x/3.x |
| 차트 | ECharts, Flot, D3 (리포트 영역) |
| 진입점 | JSP (`login.jsp`, `starter.jsp`) |

## 인프라·로컬 실행

| 방식 | 명령 | DB |
|------|------|-----|
| Docker MySQL | `docker compose up -d` + `mvn … -Denv=docker` | MySQL 8 볼륨 |
| H2 로컬 | `mvn … -Denv=local` | `target/h2-local/` 파일 DB |
| 전통 Tomcat | WAR → `/opt/apache-tomcat/webapps/bdp` | MySQL |

## 패키지 구조 (Java)

- **구 upstream Java 패키지**: 핵심 도메인·컨트롤러·데이터 프로바이더 (현재 `com.katsulabs.bi`로 대체)
- **`com.shinhan.*`**: **제거됨**. `webapp/report/` 정적 리포트 및 로그인 business_code selectbox도 제거됨

## 설정 파일

- `src/main/resources/config.properties` — JDBC, Jasypt, H2 aggregator
- ~~`login.properties` `business.codes`~~ — 제거됨 (단일 테넌트)
- `src/main/resources/spring-*.xml` — Spring / Security / MyBatis
- 프로필: `local/`, `docker/` — `maven-antrun` 으로 `target/classes` 에 복사

## 테스트

- JUnit **4.11**, `spring-test` 4.3
- 통합·E2E 테스트 거의 없음 → **TDD 도입 대상**
